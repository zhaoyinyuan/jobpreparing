### 编译注意点：
1. 模板编译方式：xelatex；实测编译时速度会变慢；
2. 编码格式：UTF-8 （中文支持）；使用texworks时右下角可以设置编码方式，或者在文件前面添加
```
%# -*- coding:utf-8 -*-
```
3. 注意补充缺失的字体；需要根据需求导入在系统中，我的模板直接改成了宋体
